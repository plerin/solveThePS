### 문제풀이 접근

- **_시간제한에 따라 적절한 알고리즘 설계_**

  - N의 범위 500 -> O(N^3)
  - N의 범위 2,000 -> O(N^2)
  - N의 범위 100,000 -> O(NlogN)
  - N의 범위 100,000,000 -> O(N)

<br>

3. 문제 해결을 위한 아이디어 찾기
